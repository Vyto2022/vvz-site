Étude de cas : Pneus — Appels saisonniers

Objectif : augmenter les appels pendant le rush de changement de pneus avec une offre saisonnière claire, un CTA “Appeler” prioritaire et une FAQ anti-friction.

Aperçu : maquette de structure (captures réelles disponibles sur demande).

Ce qui a été fait

  • Offre saisonnière : montage + équilibrage + option stockage
  • CTA “Appeler maintenant” prioritaire sur mobile
  • Blocs services : montage / équilibrage / géométrie / stockage
  • FAQ : prix, délais, RDV, conditions

Logique de conversion

  • CTA principal = appel (le plus rapide pour pneus)
  • Preuves (avis/garanties) près des CTA
  • FAQ réduit les questions répétitives
  • Chemins rapides : appel / itinéraire / horaires

Améliorations possibles

  • Ajouter un bouton “Appeler” sticky sur mobile
  • Ajouter carte + itinéraire
  • Suivre clics CTA + submits
